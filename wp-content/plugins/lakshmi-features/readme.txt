Lakshmi Features is a WordPress plugin for Lakshmi Lite theme from Webzakt.

== Description ==

Lakshmi Features is a WordPress plugin for Lakshmi Lite theme from Webzakt.
The features works only with Unyson plugin.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/lakshmi-features` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
